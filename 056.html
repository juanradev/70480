QUESTION 56

Operador === valor y tipo

You are testing the value of the following variable in JavaScript.
var height = "300";
A block of code must execute if the following conditions are true:

  • The height variable is set to 300.
  • The height variable is of type string.
You need to implement the code to run the test.
Which line of code should you use?
  • if (height == 300)
  • if (height == "300")
  • if (height ! "300")
  • if (height ! = 300)

Solución: B

nota: la verdad es que if (height == 300) daría el mismo resultado if (height == "300") ya que solo comprueba el valor
debería haber sido con:
    if (height === 300)
    if (height === "300")
Ya que con ello se verifica que valga "300" y que se de tipo string
en el ejemplo utilizo ===

Explicación:
Use == to test for equality.
Use "300" to test for the string.
