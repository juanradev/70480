QUESTION 209
Which CSS3 code segment styles the text color of EM elements to be red and semi-transparent?

  1. em {color : #ff0007f}
  2. em { color: rgba(255,0,0,0.5) }
  3. em { color: rgba(255,0,0.127) }
  4. em { color: rgba(100%,0%,0%,50%) }

Solución:
b em { color: rgba(255,0,0,0.5) }

Reference:
http://www.w3schools.com/cssref/css_colors_legal.asp

Explicación:
Hexadecimal colors: #ff0000; Pero en este caso faltaría la opacidad
RGB colors: rgb(255, 0, 0) ; Pero en este caso faltaría la opacidad
RGBA colors: rgb(255, 0, 0, 0.5) rgb + Opacidad
HSL colors: hsl(hue, saturation, lightness) hsl(125, 0%, 15%)
    Hue is a degree on the color wheel (from 0 to 360) - 0 (or 360) is red, 120 is green, 240 is blue
    saturation: 0-100%
    lightness: 0-100%
HSLA colors
     hsl(125, 0%, 15%,0.25)
    Hue is a degree on the color wheel (from 0 to 360) - 0 (or 360) is red, 120 is green, 240 is blue
    saturation: 0-100%
    lightness: 0-100%
    lpha (opacidad) 0.00 - 1
Predefined/Cross-browser color names red
With the currentcolor keyword: currentcolor
