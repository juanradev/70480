QUESTION 158
You develop an HTML5 application for a company. Employees must enter a personal identification number (PIN) in an INPUT element named SecurityCode to access their employee records.
The SecurityCode element must meet the following requirements:

  • Allow up to 6 digits.
  • Do not display numbers as they are entered.
  • Display the text Enter PIN Code before the user enters any data.
You need to implement the SecurityCode element. Which HTML markup should you add to the application?

Solución: D
<input name="SecurityCode" type="password" maxlength="6" placeholder="Introduce tu PIN CODE" >

  • Allow up to 6 digits. maxlength="6"
  • Do not display numbers as they are entered. type="password"
  • Display the text Enter PIN Code before the user enters any data.placeholder="enter...."

Explicación:
Explanation/Reference:
Explanation:
* Input Type: password
<input type="password" >defines a password field.
The characters in a password field are masked (shown as asterisks or circles).
* The placeholder attribute specifies a short hint that describes the expected value of an input field (e.g. a sample value or a short description of the expected format).
The short hint is displayed in the input field before the user enters a value.
The placeholder attribute works with the following input types: text, search, url, tel, email, and password.
