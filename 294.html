You are creating a page by using HTML5.
You need to add a video named video1 to the page. The solution must ensure that video1 can be played by browsers that can play MP4 files and browsers that can play OGG files.
Which four code blocks should you use to develop the solution? To answer, move the appropriate code blocks from the list of code blocks to the answer area and arrange them in the correct order.

Solución:


No tiene dificultad más que te acuerdes que para varios formatos cada uno va con su propio source.
y existe los atributos autplay y controls. https://www.w3schools.com/html/html5_video.asp

<video width="320" height="240" autoplay controls> <source src="https://www.w3schools.com/html/movie.mp4" type="video/mp4"> <source src="https://www.w3schools.com/html/movie.ogg" type="video/ogg"> Your browser does not support the video tag. </video>
