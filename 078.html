QUESTION 78

tags semanticos

You are modifying a blog site to improve search engine readability.
You need to group relevant page content together to maximize search engine readability.
Which tag should you use?

  • <span>
  • <nav>
  • <cable>
  • <section>

Solución: D

Explicación:
The <section> tag defines sections in a document. Such as articles, headers, footers, or any other sections of the document.
